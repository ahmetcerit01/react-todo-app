# React Todo App

A simple and modern Todo Manager built with **React + Vite**.  
Users can add, update, delete and complete tasks. All tasks are stored in **localStorage** so they remain after page refresh.

## Live Demo

https://react-todo-app.netlify.app

## Github Repository

https://github.com/ahmetcerit01/react-todo-app

## Features

- Add new tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- Tasks saved using **localStorage**
- Responsive and clean UI

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- LocalStorage API
